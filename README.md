# Osiris-bot
A simple telegram bot written in rust

```
/Start   -  Shows details about the bot
/Help    -  Display help message
/Cuss    -  Cuss you with no mercy
/Code    -  Gets the source code link
/J       -  Prints random jokes
/Short   -  Shorten a given url
```
